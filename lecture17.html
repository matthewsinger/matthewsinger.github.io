Natural numbers

Let’s think about natural numbers: 0, 1, 2, 3....

They have a certain structure to them. Let’s see if we can tease it out. What’s the base case? (0)

How do you get from one natural number to another? By adding 1.

; A Nat is one of:
; - 0
; - (add1 Nat)

The interpretation is obvious, but we need examples to make sure the definition is sound.

; A Nat is one of:
; - 0
; - (add1 Nat)
 
(define zero 0)
(define one (add1 zero))
(define two (add1 one))

Looks like we can make examples, so the DD is sound. Let’s try and write the template.

; A Nat is one of:
; - 0
; - (add1 Nat)
 
(define zero 0)
(define one (add1 zero))
(define two (add1 one))
 
; nat-temp : Nat -> ?
(define (nat-temp n)
  (cond [(zero? n) ...]
        [(positive? n) (... (nat-temp (sub1 n)))]))

Let’s try our own repeat, like replicate.

; repeat : String Nat -> String
; Repeat s n times
(define (repeat s n)
  (cond [(zero? n) ""]
        [(positive? n) (string-append s (repeat s (sub1 n)))]))
(check-expect (repeat "echo" zero) "cat")
(check-expect (repeat "echo" (add1 3)) "echoechoecho")

Without using + or *, let’s see if we can add double.

; double : Nat -> Nat
; Double n
(define (double n)
  (cond [(zero? n) 0]
        [(positive? n) (add1 (add1 (double (sub1 n))))]))
(check-expect (double zero) zero)
(check-expect (double two) 4)

Now let’s define factorial. (write definition of fact in code/the board if they don’t know it)

; ! : Nat -> Nat
; n!
(define (! n)
  (cond [(zero? n) 1]
        [(positive? n) (* n (! (sub1 n)))]))
(check-expect (! zero) one)
(check-expect (! two) two)
(check-expect (! 5) 120)

And now for an interesting one. How do we do my-even?. We can’t use modulo.

If students try to do two base cases, zero or one, write it, but say it breaks the template and show them the better way.

; my-even? : Nat -> Boolean
; Is n even?
(define (my-even? n)
  (cond [(zero? n) #t]
        [(positive? n) (not (my-even? (sub1 n)))]))
(check-expect (my-even? zero) #t)
(check-expect (my-even? one) #f)
(check-expect (my-even? two) #t)

Now let’s see if we can build up addition and multiplication (and expt if there’s time).

; plus : Nat Nat -> Nat
; a+b
(define (plus a b)
  (cond [(zero? a) b]
        [(positibe? a) (add1 (plus (sub1 a) b))]))
(check-expect (plus zero zero) zero)
(check-expect (plus zero one) one)
(check-expect (plus one zero) one)
(check-expect (plus one one) two)
(check-expect (plus 15 14) 29)
 
; times : Nat Nat -> Nat
; a*b
(define (times a b)
  (cond [(zero? a) 0]
        [(positibe? a) (plus b (times (sub1 a) b))]))
(check-expect (times zero zero) zero)
(check-expect (times zero one) zero)
(check-expect (times one zero) zero)
(check-expect (times one one) one)
(check-expect (times 5 4) 20)
 
; ^ : Nat Nat -> Nat
; a^b
(define (^ a b)
  (cond [(zero? b) 1]
        [(positive? b) (times a (^ a (sub1 b)))]))
(check-expect (^ 2 3) 8)
(check-expect (^ 2 0) 1)
(check-expect (^ 0 0) 1)
