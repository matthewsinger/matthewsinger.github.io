Sets as lists

Go through the following functions where sets of numbers are defined as lists:

; A [Set X] is a [List-of X]
 
; in? : X [Set X] -> Boolean
; Is x in s?
(define (in? x s)
  ...)
 
; add : X [Set X] -> [Set X]
; Add x to s
(define (add x s)
  ...)
 
; union : [Set X] [Set X] -> [Set X]
; The union of s1 and s2
(define (union s1 s2)
  ...)
 
; intersection : [Set X] [Set X] -> [Set X]
; The intersection of s1 and s2
(define (intersection s1 s2)
  ...)
 
; subset? : [Set X] [Set X] -> Boolean
; Is s1 a subset of s2?
(define (subset? s1 s2)
  ...)
 
; set=? : [Set X] [Set X] -> Boolean
; Is s1 equal to s2?
(define (set=? s1 s2)
  ...)

Sets as functions

Go through the same, with sets defined as a predicate. Get stuck at subset?. Use the paradoxical "does the set of all sets contain itself" to motivate omega, e.g. (self-apply self-apply) where self-apply is (λ (x) (x x)). Tease recursion with just lambda as a build up to next lecture.
