File System

Let’s thing about the file system on our computers for a second. Everything is either a file with a name and some contents or a director with a name and a list of files and directories in it.

; A Directory is a (make-folder String LoFSC)
(define-struct directory [name subfiles])
 
; A LoFSC is a [List-of FSC]
 
; An FSC (File System Component) is one of:
; - File
; - Directory
 
; A File is a (make-file String String)
(define-struct file [name content])
 
; directory-temp : Directory -> ?
(define (directory-temp d)
  (... (directory-name f) (lofsc-temp (directory-subfiles))))
 
; lofsc-temp : LoFSC -> ?
(define (lofsc-temp lofsc)
  (cond [(empty? lofsc) ...]
        [(cons? lofsc) (... (fsc-temp (first lofsc)) (lofsc-temp (rest lofsc)))]))
 
; fsc-temp : FSC -> ?
(define (fsc-temp fsc)
  (cond [(file? fsc) (file-temp fsc)]
        [(directory? fsc) (directory-temp fsc)]))
 
; file-temp : File -> ?
(define (file-temp f)
  (... (file-name f) (file-contents f)))
