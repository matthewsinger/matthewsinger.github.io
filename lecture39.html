Graph

Let’s imagine a social network where a person can amass a list of followers. How would we design that data definition?

; A Network is a [List-of Person]
; A Person is a (make-person Symbol [List-of Symbol])
(define-struct person [name contacts])
; indicating who they can contact, with no duplicats in the contacts
; and all of the contacts appearing in the network

Note that this is a kind of graph, which is another name for a network.

Define an example WITHOUT cycles, define one WITH.

We want to know if one person can contact another. People can be contacted through each other’s contacts, and can always contact themselves.

Design this function. Then test it on the non-cyclic example. Then have it break on the one with cycles.

Why is this not terminating? Because it gets stuck in a cycle. We’ll talk next class about how to fix this.
