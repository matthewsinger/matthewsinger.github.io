String split

We often want to split a string up into a list of strings and clear all of the spaces in the string. As you can imagine, the functions in 2htdp/batch-io do this.

How would we do that?

; string-split : String -> [List-of String]
(define (string-split s)
  (cond [(string? s "") '()]
        [else (cons (before-space s) (string-split (after-spaces s)))]))

Implement helpers, discuss edge cases.

Fast exponentiation

Develop the standard recursive expt function for nats.

Computing an exponent the standard way results in lots of multiplcations. What if we could reduce that?

; expt/fast : Nat Nat -> Nat
(define (expt/fast a b)
  (cond [(zero? b) 1]
        [(even? b) (local [(define a^b/2 (expt/fast a (/ b 2)))]
                     (* a^b/2 a^b/2))]
        [(odd? b) (local [(define a^sub1b/2 (expt/fast a (/ (sub1 b 2))))]
                    (* a a^sub1b/2 a^sub1b/2))]))
