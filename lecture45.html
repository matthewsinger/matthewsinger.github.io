Search by windows

When we find the root of a function on a graphing calculator, we give it a x-coordinate guaranteed to be to the left of some root and an x-coordinate guaranteed to be to the right of some root. It then approximates a root. Let’s try doing this by finding the window the x is in by determining what is an acceptable margin of error.
; PROBLEM:  The root of a function f is some x value such that f(x) = 0.
; If a function is continuous and f(lo) * f(hi) <= 0, then f has a root
; in the range [lo, hi].
 
; Design a function find-root that finds an interval of size 0.001
; that contains a root of such a function.
; Assume that f is continuous and f(lo) * f(hi) <= 0
 
(define EPSILON 0.001)
 
; find-root : Number Number Number [Number -> Number] -> Number
; Initial version (slow)
(define (find-root lo hi delta f)
(local ((define width (- hi lo))
        (define num-wins (ceiling (/ width delta)))
        (define (mk-win i) (+ lo (* i delta)))
        (define intervals (build-list num-wins mk-win))
        (define (good-win? l) (<= (* (f l) (f (+ l delta))) 0))
        (define winners (filter good-win? intervals)))
 (first winners)))
 
(check-within (find-root 0 100 EPSILON (λ (x) x)) ; x = 0
              0 EPSILON)
 
(check-within (find-root 0 100 EPSILON (λ (x) (- x 5))) ; x = 5
              5 EPSILON)
(check-within (find-root 0 100 EPSILON (λ (x) (- (sqr x) 9))) ; x = 3
              3 EPSILON)

So this seems fine, but what happens with (find-root 1 5 EPSILON sin) as we set epsilon to 0.0001 or 1e-05? Slow! Let’s do better by cutting the search for where we could be looking in half until our error is small enough.

; find-root : Number Number Number [Number -> Number] -> Number
; Find r such that (check-within (F r) 0 EPSILON) holds for r in interval
; Terminates: since each recursive call is made on a smaller interval (either [lo,mid] or [mid,hi])
; which are guaranteed to be smaller than [lo,hi]
(define (find-root lo hi delta f)
  (local ((define mid (/ (+ lo hi) 2))
          (define f@lo (f lo))
          (define f@mid (f mid))
          (define f@hi (f hi)))
    (cond [(<= (- hi lo) delta) lo] ; base case: [lo,hi] small enough we're done
          [(<= (* f@lo f@mid) 0)
           (find-root lo mid delta f)]
          [else (find-root mid hi delta f)])))
