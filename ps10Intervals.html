Intervals

Programming Language BSL

Purpose Design and work with intervals.

Setup Recall the notation for interval data, per an example from class:

; A NumericGrade is one of:
; - [90, 100]
; - [80, 89]
; - [70, 79]
; - [60, 69]
; - [0, 59]
; and represents a student's grade for a course

Be sure to use the design recipe for data when designing or given a new data definition, as described in homework 9. This will be your last reminder to do so; it is expected that you will remember this from here on in.

Exercises

Exercise 1 Some cable packages come with just a handful of channels and their HD versions. The standard definition versions are low-numbered channels (1, 2, 3, etc.) and the HD version of channel x can be found at x + 500.

Design a data definition for cable channels which includes 10 standard definition channels and their HD equivalents.

Then, design a function which either outputs "The HD version of this channel can be found at ...", filling in the dots appropriately, or the message "You are watching in HD".

Exercise 2 Design a data definition that utilizes intrervals to drive an animation. If your animation from the film festival changed behavior over time, you can retrofit it to use a proper data definition, or you can create a new one.

Remember that the signature for the function for animate takes a natural number and outputs an image, so you may need to initially transform that natural number (likely via remainder) to a number that fits in your data definition if your data definition is bounded.
