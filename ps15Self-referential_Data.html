Self-referential Data

Programming Language BSL

Purpose Design functions that involve self-referential data.

Setup If a data definition refers to itself, the template of functions that consume that data should refer to itself in the exact same spot!

Exercises

Exercise 1 Consider the following data definition:
; A MonkeyChain is a (make-monkey String String MonkeyChain)
(define-struct monkey [name c others])
; and represents a collection of monkeys where:
;   - name is the name of this monkey
;   - c is the color of this monkey
;   - others is the other monkeys it is attached to

This is inspired by the famous game Barrel of Monkeys!

What is the problem with this data definition? Describe your answer in a comment.

Hint: try defining examples of a MonkeyChain.

Exercise 2 Consider the following data definition:
; A Skyline is one of:
; - 'end
; - (make-building Positive Positive Symbol Skyline)
(define-struct building [width height color right])
; and represents either an empty skyle or a building with
; a width and height in pixels, the color of the building, and the
; rest of the skyline to the right of the building

  1. Design a function which draws a Skyline (the beside/align function will be helpful here).

  2. Design a function which determines if every building in the Skyline is a perfectly square building.

  3. Design a function which removes all 'red buildings from the skyline.

  4. Design a function which takes a Skyline and a positive number and increases the height of each building by that amount.

  5. Design a function which determines the height of the tallest building in a Skyline. If there is no building, the function should return 0.

Exercise 3 Consider the following data definition:
; A Salad is one of:
; - "lettuce"
; - (make-ingredient String Salad)
(define-struct ingredient [name more])
; and represents a collection of ingredients in a salad
Design a function which describes a salad. A salad with just lettuce should be described as "This salad has lettuce". A salad with walnuts (as the outermost ingredient) and feta should be described as "This salad has walnuts and feta and lettuce".
