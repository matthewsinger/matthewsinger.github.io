Lists

Programming Language BSL

Purpose Design data and functions around cons lists.

Setup Recall the new constants and functions from class: empty (which is the same as '()), empty?, cons, cons?, first and rest.

Below is a sample data definition using cons:
; A LoN (List of Numbers) is one of:
; - empty
; - (cons Number LoN)

Exercises

Exercise 1

  1. Develop a data definition for a list of symbols.

  2. Design a function which determines if a given symbol occurs in a list of symbols an even amount of times.

  3. Design a function which given two symbols and a list of symbols, replaces all instances of the first symbol with the second symbol.

Exercise 2

  1. Develop a data definition for a list of list of numbers.

  2. Design a function which sums every number in a list of list of numbers.

Exercise 3

  1. Design the function interleave that takes two list of numbers and produces a list of their items, alternating from each list. If the lists have different lengths, just finish with all the remaining items of the longer list.

  2. Design a function intersection that takes two lists of numbers and produces a list of numbers that appear in both lists. You can assume there are no duplicates in either list.
