I/O

Programming Language ISL

Purpose Use the file system on the computer.

Setup Don’t forget to require 2htdp/batch-io. While functions that use I/O functions can’t be directly tested, those functions should be tiny, and relegate to helpers.

Exercises

Exercise 1 Design a quiz game in which a person is presented with a series of multiple choice questions, each with three possible answers. When the program closes, it should write to a file with the contents of the questions asked, the options available for each question, the option the person chose for each question, and what the correct option actually was.

Exercise 2 Modify your program so it can read in from a library of multiple choice questions saved somewhere on the hard drive. Note: this exercise is optional and challenging. Designing a file format to read from and then encode into a proper data definition is not a trivial problem!
