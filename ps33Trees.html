Trees

Programming Language ISL+

Purpose To design functions around tree-shaped data.

Setup If a data definition recurs in multiple places, the template should reflect this.

Exercises

Exercise 1
; A Tournament is one of:
; - String
(define-struct game [score1 score2 t1 t2])
; - (make-game Number Number Tournament Tournament)
; and represents either the name of a team who entered the tournament
; or a game where two teams played, the winners of t1 and winners of t2,
; and the points t1 received (score1) and the points t2 received (score2)
; ties are impossible and all team names are unique

  1. Design the function winner that given a Tournament produces the name of the winner of the tournament.

  2. Design the fucntion biggest-victory that given a Tournament produces the largest difference between scores of a single game found in the tree.

  3. Design the function bias that given a tournament and the name of a team in the tournament, makes the minimal amount of changes to the scores to ensure that team is the winner. If a score needs to be changed, it should be made 1 bigger than the score of the other team playing that round.

Exercise 2
; A LeafyTree is one of:
; - 'leaf
(define-struct node [l r])
; - (make-node LeafyTree LeafyTree)
 
; A Path is a [List-of LR]
 
; An LR is one of:
; - 'left
; - 'right

Design a function paths that given a LeafyTree outputs the list of all Paths that represent the ways to go from the root of the tree to its leaves.
