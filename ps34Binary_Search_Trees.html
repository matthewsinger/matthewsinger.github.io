Binary Search Trees

Programming Language ISL+

Purpose To speed up programs with binary search trees.

Setup A binary search tree is one where everything in the left child is less than the value at the node, and everything in the right is greater than.

Exercises

Exercise 1 Recall the word count problem from lecture a month or so ago. Given a large text (i.e. list of strings), we want to know how many times each word appears. Design an appropriate data structure that utilizes a binary search tree to accomplish this problem.

Your solution should contain two functions: one that converts a list of strings to your data structure, and another that can search the data structure to see how many times a word was used.

Hint: the binary search tree we went over in class was just a binary search tree of numbers, but others are possible. In particular, think about what other kind of data has an ordering, and that while more than one piece of data can be stored at a node in a binary search tree, the tree only needs to be ordered according to one type of data.
