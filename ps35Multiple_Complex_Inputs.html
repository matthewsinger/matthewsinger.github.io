Multiple Complex Inputs

Programming Language ISL+

Purpose To process multiple complex inputs simultaneously.

Setup Processing two complex inputs can require the combination of both templates of the inputs, with one case for each possible combination. This is not always the case, though: functions like append are written by recurring over only one input. Be sure to pick the correct method.

Exercise 2 Design the function rotation? that takes two lists of symbols and determines if they are rotations of each other. All of the following are all rotations of each other (including itself):

  • (list 'a 'b 'c)

  • (list 'c 'a 'b)

  • (list 'b 'c 'a)

As are these:

  • (list 'a 'c 'b)

  • (list 'b 'a 'c)

  • (list 'c 'b 'a)

But no list from the first group is a rotation of the second group and vice versa.

These are also all rotations of each other:

  • (list 'a 'b 'c 'd)

  • (list 'b 'c 'd 'a)

  • (list 'c 'd 'a 'b)

  • (list 'd 'a 'b 'c)
