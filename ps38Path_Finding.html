Path Finding

Programming Language ISL+

Purpose To design a tricky recursive function.

Setup Be sure to follow the design recipe, with careful attention to template-driven helpers, the entire way through this problem; it’s a tough one.

Exercises

Exercise 1
; An Sexpr is one of:
; – Symbol
; – Lexpr
 
; A Lexpr is one of:
; – '()
; – (cons Sexpr Lexpr)
 
; A Path is a [List-of Nat] and describes how to traverse an Sexpr via indices.

For example, traversing 'a with the path '() would produce 'a.

Traversing '((a (b (c d e) f g) h i)) with the path '(0 1 1 2) would produce 'e.

; A [Maybe X] is one of:
; - #false
; - X

Design the function find-path, which given an SExpr and a symbol outputs the path to the leftmost occurence of that symbol in the SExpr. If that symbol is not in the sexpr, return #false.
