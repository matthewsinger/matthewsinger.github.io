Binary search

Programming Language ISL+

Purpose To use binary search to find a 0.

Setup Binary search is a general algorithmic principal which cuts the potential area for something to be in in half.

Exercises

Exercise 1 Use binary search to design a function which takes an ordered list of strictly increasing integers and returns the index of 0 in the list, or #f if 0 is not present.
