Animations

Programming Language BSL

Purpose Become comfortable with functions that produce images, write animations

Setup To run an animation, we enter (animate a-function) into the interactions window. a-function has to have the signature

; a-function : Nat -> Image

Running an animation is not a replacement for testing the function that it uses!

Recall the remainder function, which will be useful for looping animations, and the place-image function, which will be useful for placing images on backgrounds. empty-scene is useful for creating backgrounds. Look these functions and any others you may need up in the docs.

Use constants when appropriate.

Be sure to use the design recipe for functions, as described in homework 3. This will be your last reminder to do so; it is expected that you will remember this from here on in.

Don’t forget to add (require 2htdp/image) and (require 2htdp/universe) at the top of your file and to be sure you’re using BSL. This will be your last reminder to do so; it is expected that you will remember this from here on in.

Exercises

Exercise 1 Design an animation called growing, which grows a circle placed in the middle of the screen from non-existant to its border touching the edge of the screen. This animation should loop.

Exercise 2 Design an animation called growing2, which is the same as growing, except the circle grows twice as fast.

Exercise 3 Design an animation called get-me-out-of-here, which moves a dot from the bottom left hand corner of the screen to and then past the upper right hand corner.

Exercise 4 Design an animation called crowded, which moves a triangle from right to left and a square from top to bottom. This should happen on a rectangle, and each should loop back back to the right and top, respectively, when it reaches the left and bottom, respectively.

Exercise 5 Design an animation called teleprompter which moves a block of text from the bottom of the screen up, allowing text that didn’t fit to now be seen.

You will need to use the text function, as well as have line breaks in the string you give to the text function. To put a line break in a string in BSL, we use the escape character \ followed by an n (which is short for new line). For example, "line 1\nline 2". Run (text "line 1\nline 2" 24 "olive") in the interactions window to see how this works.

Your animation text can be whatever news story you’d like to pen. Have some fun with it!

Hint: defining your text as a constant will make your code cleaner, as will defining the height of the image the text function produces as a constant.
