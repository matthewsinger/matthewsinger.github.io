Grab bag

Programming Language BSL

Purpose Combine booleans with and, or, and not, get practice with error, symbols and random

Setup Remember, a symbol looks like a string, but only has one opening quote, such as 'example.

check-error and check-random can be used to test functions that produce errors and that use random numbers, respectively.

Exercises

Exercise 1 Design a function between-and-divides? that takes three positive integers, a, b, and c, and determines if a is both between b and c and divides exactly one of them. Note that b could be smaller or bigger than c!

Exercise 2 Design a function numeric? which determines if a string is a string of a number. Note that this only applies to literal number strings, like "0", and not their spellings, like "zero".

Exercise 3 Design a function primitive->type that takes any primitive value and outputs its type as a symbol (e.g. 'number, 'image, etc.). We will call the input type to this function Any, meaning any kind of value. For now, all of our values are primitives, as we only know about numbers, strings, images, booleans, and symbols.

Exercise 4 Design a function primitive-equal? that determines if two primitive values are equal. If two primitives of different types (such as a number and a string) are passed to the function, it should throw an error. You should not use equal? in this problem.

Exercise 5 Your friend Persephone has written a function that takes a positive integer and determines if a single randomly generated number less than it is both larger than 7 and has a remainder of 3 when divided by 7.

; persephone : Number -> Boolean
; Is a random number less than n greater than 7 and
; has a remainder of 3 when divided by 7?
(define (persephone n)
  (and (> (random n) 7) (= 3 (remainder (random n) 7))))

Her logic, however, is a bit off. Leave a comment explaining why Persephone’s code isn’t doing exactly what she thinks it’s doing, correct the code, and add tests.

Hint: Persephone is correctly comparing numbers to each other and is checking the remainder correctly, too. Neither of those places is where her code is going wrong. To fix it, you will need a helper function that will look an awful lot like the original one.

Exercise 6 Design an animation quadrants in which a dot randomly teleports between four quadrants of a square. Note that the dot should only ever appear in the exact center of these quadrants.

Hint: you may not want to use the number passed to quadrants at all. Arguments passed to functions that aren’t used are often called _ to indicate to the reader it isn’t being used.
